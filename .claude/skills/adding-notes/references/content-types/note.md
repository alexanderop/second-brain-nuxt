# Personal Notes

Personal notes for synthesized thoughts and ideas.

## Detection

- No URL provided
- User explicitly requests a note
- Type override: `note`

## Frontmatter

```yaml
---
title: "Note Title"
type: note
tags:
  - topic-1
  - topic-2
summary: "Core argument: What claim does this note make? (assertion, not description)"
date: YYYY-MM-DD  # Use today's date when creating the note (must not be in future)
---
```

## Body Template

```markdown
[Synthesized thoughts and ideas]

## Connections

[Each link must explain WHY you're connecting these notes. Minimum 2 links required.]

- [[concept-1]] - [1-sentence explanation of the relationship]
- [[concept-2]] - [1-sentence explanation of the relationship]
```

## Notes

- No external URL required
- Focus on personal synthesis, not source summary
- Use wiki-links to connect to related notes
