# Evergreen Notes

Foundational reference notes synthesizing multiple sources.

## Detection

- Type override: `evergreen`
- User wants to create a foundational concept note

## Frontmatter

```yaml
---
title: "Concept Name"
type: evergreen
tags:
  - topic-1
  - topic-2
summary: "Core definition of this concept"
date: 2026-01-01
---
```

## Body Template

```markdown
## Definition

[Core concept explained in your own words]

## Key Principles

- Principle 1
- Principle 2

## Sources

Synthesized from [[source-1]], [[source-2]], and [[source-3]].
```

## Notes

- Evergreen notes should be **timeless** - not tied to a specific source
- Synthesize multiple sources into a cohesive understanding
- Use the Sources section to link to original materials
- Write in your own words, not quotes
