# Introduction

Second Brain is a personal knowledge base for capturing and connecting content using Zettelkasten-style wiki-links. Built with Nuxt 4 and @nuxt/content v3, it provides a beautiful interface for organizing podcasts, articles, books, and your own notes.

## Prerequisites

Before you begin, ensure you have the following installed:

- **Node.js 24.x** or higher
- **pnpm 10+** - This project uses pnpm workspaces

::alert{type="info"}
You can install pnpm globally with `npm install -g pnpm` or via [Corepack](https://pnpm.io/installation#using-corepack): `corepack enable && corepack prepare pnpm@latest --activate`
::

## Fork and Clone

1. **Fork the repository** on GitHub by clicking the "Fork" button at [github.com/alexanderop/second-brain-nuxt](https://github.com/alexanderop/second-brain-nuxt)

2. **Clone your fork** to your local machine:

```bash
git clone https://github.com/YOUR_USERNAME/second-brain-nuxt.git
cd second-brain-nuxt
```

3. **Add the upstream remote** (optional, for staying up to date):

```bash
git remote add upstream https://github.com/alexanderop/second-brain-nuxt.git
```

## Environment Setup

This project requires minimal configuration. There are no required environment variables for basic development.

If you want to customize the site for your own use, edit `apps/web/site.config.ts`:

```typescript
export const siteConfig = {
  name: 'My Second Brain',
  description: 'My personal knowledge base',
  // ... other options
}
```

See the [Site Configuration](/customization/site-config) guide for all available options.

## First Run

1. **Install dependencies**:

```bash
pnpm install
```

2. **Start the development server**:

```bash
pnpm dev
```

3. **Open your browser** and visit [http://localhost:3000](http://localhost:3000)

You should see the Second Brain homepage with sample content.

## Verification Steps

To verify everything is working correctly:

::list{type="success"}
- The homepage loads at `localhost:3000`
- You can navigate to different pages (Books, Podcasts, Graph, etc.)
- The search modal opens when pressing `Cmd+K` (or `Ctrl+K` on Windows/Linux)
- Content pages display with wiki-links highlighted
::

## Available Scripts

From the root of the monorepo, you have access to these commands:

| Command | Description |
|---------|-------------|
| `pnpm dev` | Start the web app at localhost:3000 |
| `pnpm dev:docs` | Start the docs site at localhost:3001 |
| `pnpm build` | Build the web app for production |
| `pnpm lint:fix` | Auto-fix linting issues |
| `pnpm typecheck` | Verify type safety |
| `pnpm test:unit` | Run fast unit tests |
